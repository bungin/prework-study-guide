GIT:

    The git command must precede every Git statement.

    The checkout command is to move the working branch to a new branch. The -b flag creates a new branch.

    The feature/starter-code is the name of the new branch.
    We have added the prefix, feature, to indicate that it is a feature.
    It is good practice to name the branches for the feature that will be developed by them to help indicate the purpose of each branch.

     The add command adds modifications in the current working branch to the staging area.
